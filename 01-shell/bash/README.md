# Bash

## Overview

Bash (Bourne Again Shell) is the default command-line shell on many Linux distributions. It provides an interface between the user and the operating system, allowing commands to be executed, files to be managed, processes to be controlled, and administrative tasks to be automated.

This module documents the Bash concepts I learned through daily Linux usage, OverTheWire Bandit, AWS EC2 practice, and personal Linux virtual machines.

## Learning Objectives

After completing this module I was able to:

* Navigate the filesystem using the command line.
* Create, move, copy and remove files and directories.
* Inspect file contents.
* Search for files.
* Use the shell history.
* Understand the difference between built-in commands and external commands.
* Work efficiently using tab completion.

## Key Concepts

### What is a Shell?

A shell is a command interpreter that allows users to interact with the Linux operating system.

### What is Bash?

Bash is one of the most widely used Linux shells and is the default shell on many distributions.

### Terminal vs Shell

The terminal is the application you interact with.

Bash is the program running inside the terminal that interprets your commands.
